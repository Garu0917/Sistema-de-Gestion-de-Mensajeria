# 🚚 Sistema de Gestión de Mensajería

Este proyecto tiene como objetivo gestionar la distribución de guías de mensajería, permitiendo el control de asignación, seguimiento y estado de entregas.

## 🧩 Metodología

Se implementó la metodología ágil Scrum utilizando Jira, donde se realizó:

- Construcción de épicas
- Creación de release (versión 1.0)
- Organización del backlog
- Planificación y ejecución de un sprint de 15 días
- Definición de historias de usuario con subtareas

## 📋 Historias de Usuario

Las historias fueron definidas bajo el formato:

> Yo como [rol] necesito [acción] para [beneficio]

Incluyendo subtareas en diferentes estados:
- En curso
- En pruebas
- Terminadas

## ⚙️ Control de Versiones

Se utilizó Git y GitHub para el control de versiones con la siguiente estructura de ramas:

- `main`: rama principal
- `develop`: rama de desarrollo
